## Web の基礎知識

### 概要

#### Q&A

##### Web の基礎知識ってなに？

Web 開発を行う上で必要になる基礎的な知識のことです。例えばサーバーやブラウザ、プログラミング言語とその役割、データベースや HTTP などの通信などです。

##### 他分野とのつながり

フロントエンド開発では主に HTML, CSS, Javascript の言語の他にも HTTP,ファイル,サーバーなどと多くの分野と関連しています。
